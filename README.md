# 💊 MedTrack

Most medicine reminder apps just send a notification… and hope for the best.  
**MedTrack** makes sure someone actually notices when a dose is missed.

MedTrack is a **family-centered medication management app** that helps people take medicines on time with the support of their loved ones.

---

## 👨‍👩‍👧 Family Hub System

MedTrack introduces a **Family Hub** where families can manage medications together.

Users can:
- Create a **Family Hub**
- Join using a **unique hub code**
- Send join requests
- Hub creator approves or declines members

This creates a **shared care system instead of individual reminders**.

The dashboard acts as the **central hub for medication tracking**.

It shows:

- Family Hub name
- Hub code for inviting others
- All medication cards
- Medication status for the day

This gives the entire family a **clear view of medication activity**.

# 💊 Medication Cards

Every medicine is stored as a **Medication Card**.

Each card contains:

- Medicine name
- Dosage
- Time to take
- Assigned family member
- Optional remarks
- Optional image of the medicine

Only the **creator of the card can edit or delete it**, preventing accidental changes.

# 🔔 Smart Reminder System

MedTrack ensures medicines are not silently missed.

Reminder flow:

1️⃣ Notification at the scheduled time  
2️⃣ Reminder again after **5 minutes**  
3️⃣ After **10 minutes**, **family members are notified**

This creates a **support network for medication care**.

---

# 🤖 CareMate Chatbot

MedTrack includes **CareMate**, a simple assistant for medication-related questions.

CareMate helps users with:

- General medicine guidance
- Understanding medication schedules
- Answering common medication doubts

⚠️ *CareMate provides general information and does not replace professional medical advice.*
